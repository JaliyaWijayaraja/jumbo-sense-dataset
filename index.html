Seismic Signal Dataset of Sri Lankan Elephant Footfalls

About the Dataset

The JumboSense dataset contains seismic recordings from Sri Lankan elephants (Elephas maximus maximus) and non-elephant sources (e.g., cattle, humans, vehicles). Data were collected in controlled and naturalistic environments across various locations in Sri Lanka to support wildlife monitoring and signal classification research.

Contents

  • Elephant and non-elephant seismic signals
  • Controlled and naturalistic observation paradigms
  • Locations: Kaudulla, Minneriya, Pinnawala, Dehiwala, and more
  • Formatted as .csv with time and amplitude fields

Applications

  • Animal movement classification using ML
  • Human-elephant conflict (HEC) early warning systems
  • Embedded sensing and signal processing
  • Sensor fusion and ecological research
